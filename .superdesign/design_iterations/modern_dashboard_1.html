Linear Regression

A simple baseline that fits a linear trend to the data. Best for data with a clear monotonic trend and no strong seasonality.

Model Parameters
Fits a least-squares linear trend: ŷ = β₀ + β₁t. Best for monotonically trending data without seasonality.
ARIMA(p,d,q): p lagged observations, d-order differencing, q moving average terms. Use AIC/BIC to tune.
SARIMA(p,d,q)(P,D,Q)[m]. Set m=12 for monthly, m=4 for quarterly, m=7 for weekday-weekly.
VAR(p) models k time series jointly. Each variable is expressed as a linear function of past values of all other variables.
Structural time series via Kalman filter. Decomposes into level, trend, and seasonal components with separate noise terms.
Extracts k common latent factors from multivariate data via EM. Ideal for macroeconomic nowcasting with many correlated indicators.
